# SIMANDOK

Sistem Informasi Manajemen Dokumen - DKSHR Universitas Andalas

## Tech Stack

- **Frontend**: React 18, TypeScript, Vite, MUI, Redux Toolkit
- **Backend**: Node.js 18, Express.js, TypeScript, Sequelize
- **Database**: MySQL 8.0
- **Storage**: Microsoft OneDrive (Graph API)
- **Email**: Nodemailer (batch reminder only)

## Quick Start

### Prerequisites
- Node.js 18+
- MySQL 8.0
- Docker (optional)

### Development

```bash
# Backend
cd backend
npm install
npm run dev

# Frontend
cd frontend
npm install
npm run dev
```

### Docker

```bash
docker-compose up -d
```
